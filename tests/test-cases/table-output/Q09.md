## Question

Return a table comparing promo versus no-promo orders, with columns promo_bucket, orders, net_revenue, and avg_order_net_value.

## Reference SQL

```sql
WITH order_promo_flags AS (SELECT order_id, MAX(CASE WHEN promo_id <> 'NO_PROMO' OR COALESCE(promo_id_2, '') NOT IN ('', 'NO_PROMO') THEN 1 ELSE 0 END) AS used_promo FROM order_items GROUP BY 1), order_net_values AS (SELECT order_id, SUM(quantity * unit_price - discount_amount) AS order_net_value FROM order_items GROUP BY 1) SELECT CASE WHEN p.used_promo = 1 THEN 'used_promo' ELSE 'no_promo' END AS promo_bucket, COUNT(*) AS orders, ROUND(SUM(v.order_net_value), 2) AS net_revenue, ROUND(AVG(v.order_net_value), 2) AS avg_order_net_value FROM order_promo_flags p JOIN order_net_values v USING (order_id) GROUP BY 1 ORDER BY orders DESC, promo_bucket
```
