## Question

Return a table of monthly delivered order counts and delivered net revenue for every month in 2016, with columns order_month, delivered_orders, and delivered_net_revenue.

## Reference SQL

```sql
SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, COUNT(DISTINCT o.order_id) FILTER (WHERE o.order_status = 'delivered') AS delivered_orders, ROUND(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) WHERE CAST(o.order_date AS DATE) >= DATE '2016-01-01' AND CAST(o.order_date AS DATE) < DATE '2017-01-01' GROUP BY 1 ORDER BY 1
```
