## Question

Return a table of delivered net revenue by gender and device_type, with columns gender, device_type, delivered_orders, and delivered_net_revenue.

## Reference SQL

```sql
SELECT c.gender, o.device_type, COUNT(DISTINCT o.order_id) FILTER (WHERE o.order_status = 'delivered') AS delivered_orders, ROUND(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END), 2) AS delivered_net_revenue FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) GROUP BY 1, 2 ORDER BY gender, delivered_net_revenue DESC, device_type
```
