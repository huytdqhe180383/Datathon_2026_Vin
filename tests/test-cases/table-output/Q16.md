## Question

Return a table of delivered average order value by age_group and payment_method, with columns age_group, payment_method, delivered_orders, and avg_delivered_order_value.

## Reference SQL

```sql
WITH delivered_order_values AS (SELECT c.age_group, o.payment_method, o.order_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS delivered_order_net_value FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1, 2, 3) SELECT age_group, payment_method, COUNT(*) AS delivered_orders, ROUND(AVG(delivered_order_net_value), 2) AS avg_delivered_order_value FROM delivered_order_values GROUP BY 1, 2 ORDER BY age_group, avg_delivered_order_value DESC, payment_method
```
