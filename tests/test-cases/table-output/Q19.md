## Question

Return a table of order-size buckets, with columns unit_bucket, orders, avg_order_net_value, and total_net_revenue.

## Reference SQL

```sql
WITH order_units AS (SELECT o.order_id, SUM(oi.quantity) AS total_units, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS order_net_value FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1) SELECT CASE WHEN total_units = 1 THEN '1' WHEN total_units BETWEEN 2 AND 3 THEN '2-3' WHEN total_units BETWEEN 4 AND 6 THEN '4-6' WHEN total_units BETWEEN 7 AND 10 THEN '7-10' ELSE '11+' END AS unit_bucket, COUNT(*) AS orders, ROUND(AVG(order_net_value), 2) AS avg_order_net_value, ROUND(SUM(order_net_value), 2) AS total_net_revenue FROM order_units GROUP BY 1 ORDER BY CASE unit_bucket WHEN '1' THEN 1 WHEN '2-3' THEN 2 WHEN '4-6' THEN 3 WHEN '7-10' THEN 4 ELSE 5 END
```
