## Question

Return a table of zip-code loyalty metrics for zip codes with at least 50 ordering customers, with columns zip, ordering_customers, repeat_customers, repeat_customer_rate_pct, and avg_orders_per_customer.

## Reference SQL

```sql
WITH zip_customer_orders AS (SELECT zip, customer_id, COUNT(*) AS order_count FROM orders GROUP BY 1, 2) SELECT zip, COUNT(*) AS ordering_customers, SUM(CASE WHEN order_count > 1 THEN 1 ELSE 0 END) AS repeat_customers, ROUND(100.0 * SUM(CASE WHEN order_count > 1 THEN 1 ELSE 0 END) / COUNT(*), 4) AS repeat_customer_rate_pct, ROUND(AVG(order_count), 4) AS avg_orders_per_customer FROM zip_customer_orders GROUP BY 1 HAVING COUNT(*) >= 50 ORDER BY repeat_customer_rate_pct DESC, zip LIMIT 20
```
