## Question

Return a table of quarterly delivered performance, with columns year_quarter, delivered_orders, delivered_units, and delivered_net_revenue.

## Reference SQL

```sql
SELECT strftime(CAST(order_date AS DATE), '%Y') || '-Q' || CAST(quarter(CAST(order_date AS DATE)) AS VARCHAR) AS year_quarter, COUNT(DISTINCT o.order_id) AS delivered_orders, SUM(oi.quantity) AS delivered_units, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY 1
```
