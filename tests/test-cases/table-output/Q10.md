## Question

Return a table with the top 20 product_id values by delivered net revenue, including columns product_id, delivered_units, delivered_net_revenue, and avg_unit_price.

## Reference SQL

```sql
SELECT oi.product_id, SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity ELSE 0 END) AS delivered_units, ROUND(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END), 2) AS delivered_net_revenue, ROUND(AVG(CASE WHEN o.order_status = 'delivered' THEN oi.unit_price END), 2) AS avg_unit_price FROM order_items oi JOIN orders o USING (order_id) GROUP BY 1 HAVING SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity ELSE 0 END) > 0 ORDER BY delivered_net_revenue DESC, product_id LIMIT 20
```
