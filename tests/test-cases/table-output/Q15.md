## Question

Return a table with the top 10 customers by lifetime net revenue, including columns customer_id, acquisition_channel, age_group, order_count, and lifetime_net_revenue.

## Reference SQL

```sql
WITH customer_revenue AS (SELECT o.customer_id, COUNT(DISTINCT o.order_id) AS order_count, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS lifetime_net_revenue FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1) SELECT r.customer_id, c.acquisition_channel, c.age_group, r.order_count, r.lifetime_net_revenue FROM customer_revenue r JOIN customers c USING (customer_id) ORDER BY r.lifetime_net_revenue DESC, r.customer_id LIMIT 10
```
