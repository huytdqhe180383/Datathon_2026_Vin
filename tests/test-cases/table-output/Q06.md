## Question

Return a table showing the discount summary by device_type, with columns device_type, gross_revenue, total_discount_amount, and discount_rate_pct.

## Reference SQL

```sql
SELECT o.device_type, ROUND(SUM(oi.quantity * oi.unit_price), 2) AS gross_revenue, ROUND(SUM(oi.discount_amount), 2) AS total_discount_amount, ROUND(100.0 * SUM(oi.discount_amount) / NULLIF(SUM(oi.quantity * oi.unit_price), 0), 4) AS discount_rate_pct FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1 ORDER BY discount_rate_pct DESC, device_type
```
