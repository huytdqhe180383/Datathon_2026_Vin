## Question

Return a table summarizing order outcomes by order_source, with columns order_source, total_orders, delivered_orders, returned_orders, cancelled_orders, and delivered_rate_pct.

## Reference SQL

```sql
SELECT order_source, COUNT(*) AS total_orders, SUM(CASE WHEN order_status = 'delivered' THEN 1 ELSE 0 END) AS delivered_orders, SUM(CASE WHEN order_status = 'returned' THEN 1 ELSE 0 END) AS returned_orders, SUM(CASE WHEN order_status = 'cancelled' THEN 1 ELSE 0 END) AS cancelled_orders, ROUND(100.0 * SUM(CASE WHEN order_status = 'delivered' THEN 1 ELSE 0 END) / COUNT(*), 4) AS delivered_rate_pct FROM orders GROUP BY 1 ORDER BY delivered_rate_pct DESC, order_source
```
