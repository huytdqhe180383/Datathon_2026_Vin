## Question

Return a table showing repeat-customer performance by acquisition_channel, with columns acquisition_channel, ordering_customers, repeat_customers, and repeat_customer_rate_pct.

## Reference SQL

```sql
WITH customer_orders AS (SELECT c.acquisition_channel, o.customer_id, COUNT(*) AS order_count FROM customers c JOIN orders o USING (customer_id) GROUP BY 1, 2) SELECT acquisition_channel, COUNT(*) AS ordering_customers, SUM(CASE WHEN order_count > 1 THEN 1 ELSE 0 END) AS repeat_customers, ROUND(100.0 * SUM(CASE WHEN order_count > 1 THEN 1 ELSE 0 END) / COUNT(*), 4) AS repeat_customer_rate_pct FROM customer_orders GROUP BY 1 ORDER BY repeat_customer_rate_pct DESC, acquisition_channel
```
