## Question

Return a table with the top 10 acquisition channels ranked by delivered net revenue, including columns acquisition_channel, delivered_orders, delivered_net_revenue, and avg_delivered_order_value.

## Reference SQL

```sql
WITH delivered_order_values AS (SELECT o.order_id, c.acquisition_channel, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS delivered_order_net_value FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1, 2) SELECT acquisition_channel, COUNT(*) AS delivered_orders, ROUND(SUM(delivered_order_net_value), 2) AS delivered_net_revenue, ROUND(AVG(delivered_order_net_value), 2) AS avg_delivered_order_value FROM delivered_order_values GROUP BY 1 ORDER BY delivered_net_revenue DESC, acquisition_channel LIMIT 10
```
