## Question

Return a table showing signup-month cohorts, with columns signup_month, customers_signed_up, customers_who_ordered, total_orders, and delivered_orders.

## Reference SQL

```sql
WITH cohort_orders AS (SELECT c.customer_id, strftime(CAST(c.signup_date AS DATE), '%Y-%m') AS signup_month, COUNT(DISTINCT o.order_id) AS total_orders, COUNT(DISTINCT o.order_id) FILTER (WHERE o.order_status = 'delivered') AS delivered_orders FROM customers c LEFT JOIN orders o USING (customer_id) GROUP BY 1, 2) SELECT signup_month, COUNT(*) AS customers_signed_up, SUM(CASE WHEN total_orders > 0 THEN 1 ELSE 0 END) AS customers_who_ordered, SUM(total_orders) AS total_orders, SUM(delivered_orders) AS delivered_orders FROM cohort_orders GROUP BY 1 ORDER BY signup_month
```
