## Question

Return a table showing monthly promo adoption, with columns order_month, total_orders, promo_orders, and promo_order_rate_pct.

## Reference SQL

```sql
WITH order_promo_flags AS (SELECT order_id, MAX(CASE WHEN promo_id <> 'NO_PROMO' OR COALESCE(promo_id_2, '') NOT IN ('', 'NO_PROMO') THEN 1 ELSE 0 END) AS used_promo FROM order_items GROUP BY 1) SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, COUNT(*) AS total_orders, SUM(CASE WHEN p.used_promo = 1 THEN 1 ELSE 0 END) AS promo_orders, ROUND(100.0 * SUM(CASE WHEN p.used_promo = 1 THEN 1 ELSE 0 END) / COUNT(*), 4) AS promo_order_rate_pct FROM orders o JOIN order_promo_flags p USING (order_id) GROUP BY 1 ORDER BY 1
```
