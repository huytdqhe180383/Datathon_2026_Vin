## Question

Return a table showing monthly new versus repeat ordering customers, with columns order_month, new_ordering_customers, repeat_ordering_customers, and total_orders.

## Reference SQL

```sql
WITH monthly_customer_orders AS (SELECT strftime(CAST(order_date AS DATE), '%Y-%m') AS order_month, customer_id, COUNT(*) AS orders_in_month FROM orders GROUP BY 1, 2), customer_first_month AS (SELECT customer_id, MIN(strftime(CAST(order_date AS DATE), '%Y-%m')) AS first_order_month FROM orders GROUP BY 1) SELECT m.order_month, SUM(CASE WHEN m.order_month = f.first_order_month THEN 1 ELSE 0 END) AS new_ordering_customers, SUM(CASE WHEN m.order_month > f.first_order_month THEN 1 ELSE 0 END) AS repeat_ordering_customers, SUM(m.orders_in_month) AS total_orders FROM monthly_customer_orders m JOIN customer_first_month f USING (customer_id) GROUP BY 1 ORDER BY 1
```
