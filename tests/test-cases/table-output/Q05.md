## Question

Return a table summarizing customer activity by age_group, with columns age_group, customers, customers_with_delivered_orders, delivered_orders, delivered_net_revenue, and avg_delivered_revenue_per_active_customer.

## Reference SQL

```sql
WITH customer_delivered_revenue AS (SELECT c.customer_id, c.age_group, COUNT(DISTINCT o.order_id) FILTER (WHERE o.order_status = 'delivered') AS delivered_orders, SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END) AS delivered_net_revenue FROM customers c LEFT JOIN orders o USING (customer_id) LEFT JOIN order_items oi USING (order_id) GROUP BY 1, 2) SELECT age_group, COUNT(*) AS customers, SUM(CASE WHEN delivered_orders > 0 THEN 1 ELSE 0 END) AS customers_with_delivered_orders, SUM(delivered_orders) AS delivered_orders, ROUND(SUM(COALESCE(delivered_net_revenue, 0)), 2) AS delivered_net_revenue, ROUND(AVG(CASE WHEN delivered_orders > 0 THEN delivered_net_revenue END), 2) AS avg_delivered_revenue_per_active_customer FROM customer_delivered_revenue GROUP BY 1 ORDER BY delivered_net_revenue DESC, age_group
```
