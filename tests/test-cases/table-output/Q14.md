## Question

Return a table showing the average number of days between the first and second order by acquisition_channel, with columns acquisition_channel, repeat_customers, and avg_days_first_to_second_order.

## Reference SQL

```sql
WITH sequenced_orders AS (SELECT o.customer_id, c.acquisition_channel, CAST(o.order_date AS DATE) AS order_date, ROW_NUMBER() OVER (PARTITION BY o.customer_id ORDER BY CAST(o.order_date AS DATE), o.order_id) AS rn FROM orders o JOIN customers c USING (customer_id)), first_two AS (SELECT customer_id, acquisition_channel, MAX(CASE WHEN rn = 1 THEN order_date END) AS first_order_date, MAX(CASE WHEN rn = 2 THEN order_date END) AS second_order_date FROM sequenced_orders WHERE rn <= 2 GROUP BY 1, 2) SELECT acquisition_channel, COUNT(*) FILTER (WHERE second_order_date IS NOT NULL) AS repeat_customers, ROUND(AVG(date_diff('day', first_order_date, second_order_date)) FILTER (WHERE second_order_date IS NOT NULL), 4) AS avg_days_first_to_second_order FROM first_two GROUP BY 1 ORDER BY avg_days_first_to_second_order, acquisition_channel
```
