## Question

Plot monthly promo versus no-promo order counts. Return a stacked bar chart using order_month, promo_bucket, and order_count.

## Reference SQL

```sql
WITH order_promo_flags AS (SELECT order_id, MAX(CASE WHEN promo_id <> 'NO_PROMO' OR COALESCE(promo_id_2, '') NOT IN ('', 'NO_PROMO') THEN 1 ELSE 0 END) AS used_promo FROM order_items GROUP BY 1) SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, CASE WHEN p.used_promo = 1 THEN 'used_promo' ELSE 'no_promo' END AS promo_bucket, COUNT(*) AS order_count FROM orders o JOIN order_promo_flags p USING (order_id) GROUP BY 1, 2 ORDER BY 1, 2
```
