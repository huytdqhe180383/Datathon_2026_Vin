## Question

Plot the distribution of delivered_order_net_value by payment_method. Return a box plot using payment_method and delivered_order_net_value.

## Reference SQL

```sql
WITH delivered_order_values AS (SELECT o.order_id, o.payment_method, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS delivered_order_net_value FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1, 2) SELECT payment_method, delivered_order_net_value FROM delivered_order_values ORDER BY payment_method, delivered_order_net_value
```
