## Question

Plot delivered_rate_pct by order_source. Return a bar chart sorted from lowest to highest delivered rate.

## Reference SQL

```sql
SELECT order_source, ROUND(100.0 * SUM(CASE WHEN order_status = 'delivered' THEN 1 ELSE 0 END) / COUNT(*), 4) AS delivered_rate_pct FROM orders GROUP BY 1 ORDER BY delivered_rate_pct, order_source
```
