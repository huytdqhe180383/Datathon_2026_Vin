## Question

Plot delivered net revenue by day of week. Return a bar chart sorted Monday through Sunday.

## Reference SQL

```sql
SELECT strftime(CAST(o.order_date AS DATE), '%A') AS weekday_name, date_part('dow', CAST(o.order_date AS DATE)) AS weekday_index, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1, 2 ORDER BY weekday_index
```
