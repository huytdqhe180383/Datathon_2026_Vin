## Question

Plot the top 15 zip codes by delivered net revenue. Return a horizontal bar chart using zip and delivered_net_revenue.

## Reference SQL

```sql
SELECT o.zip, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY delivered_net_revenue DESC, zip LIMIT 15
```
