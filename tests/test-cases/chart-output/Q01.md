## Question

Plot monthly delivered net revenue over time. Return a line chart with order_month on the x-axis and delivered_net_revenue on the y-axis.

## Reference SQL

```sql
SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, ROUND(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1 ORDER BY 1
```
