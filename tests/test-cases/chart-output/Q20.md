## Question

Plot delivered_rate_pct against avg_delivered_order_value by acquisition_channel. Return a scatter plot using acquisition_channel labels.

## Reference SQL

```sql
WITH channel_orders AS (SELECT c.acquisition_channel, COUNT(*) AS total_orders, SUM(CASE WHEN o.order_status = 'delivered' THEN 1 ELSE 0 END) AS delivered_orders FROM customers c JOIN orders o USING (customer_id) GROUP BY 1), channel_delivered_values AS (SELECT c.acquisition_channel, o.order_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS delivered_order_net_value FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1, 2) SELECT co.acquisition_channel, ROUND(100.0 * co.delivered_orders / co.total_orders, 4) AS delivered_rate_pct, ROUND(AVG(cd.delivered_order_net_value), 2) AS avg_delivered_order_value FROM channel_orders co JOIN channel_delivered_values cd USING (acquisition_channel) GROUP BY 1, 2 ORDER BY delivered_rate_pct DESC, acquisition_channel
```
