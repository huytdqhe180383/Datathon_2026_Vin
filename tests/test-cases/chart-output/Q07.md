## Question

Plot repeat_customer_rate_pct by acquisition_channel. Return a bar chart with acquisition_channel on the x-axis.

## Reference SQL

```sql
WITH customer_orders AS (SELECT c.acquisition_channel, o.customer_id, COUNT(*) AS order_count FROM customers c JOIN orders o USING (customer_id) GROUP BY 1, 2) SELECT acquisition_channel, ROUND(100.0 * SUM(CASE WHEN order_count > 1 THEN 1 ELSE 0 END) / COUNT(*), 4) AS repeat_customer_rate_pct FROM customer_orders GROUP BY 1 ORDER BY repeat_customer_rate_pct DESC, acquisition_channel
```
