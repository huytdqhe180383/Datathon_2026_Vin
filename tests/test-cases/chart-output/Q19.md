## Question

Plot monthly delivered net revenue for the top 5 acquisition channels. Return a multi-series line chart using order_month, acquisition_channel, and delivered_net_revenue.

## Reference SQL

```sql
WITH channel_revenue AS (SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, c.acquisition_channel, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS delivered_net_revenue FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1, 2), top_channels AS (SELECT acquisition_channel FROM (SELECT acquisition_channel, SUM(delivered_net_revenue) AS total_revenue FROM channel_revenue GROUP BY 1 ORDER BY total_revenue DESC, acquisition_channel LIMIT 5)) SELECT cr.order_month, cr.acquisition_channel, ROUND(cr.delivered_net_revenue, 2) AS delivered_net_revenue FROM channel_revenue cr JOIN top_channels tc USING (acquisition_channel) ORDER BY cr.order_month, cr.acquisition_channel
```
