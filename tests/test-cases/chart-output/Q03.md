## Question

Plot delivered net revenue by acquisition_channel. Return a bar chart sorted from highest to lowest revenue.

## Reference SQL

```sql
SELECT c.acquisition_channel, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY delivered_net_revenue DESC, acquisition_channel
```
