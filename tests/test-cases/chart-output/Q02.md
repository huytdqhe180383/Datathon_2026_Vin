## Question

Plot monthly delivered and returned order counts over time. Return a multi-series line chart with order_month on the x-axis.

## Reference SQL

```sql
SELECT strftime(CAST(order_date AS DATE), '%Y-%m') AS order_month, SUM(CASE WHEN order_status = 'delivered' THEN 1 ELSE 0 END) AS delivered_orders, SUM(CASE WHEN order_status = 'returned' THEN 1 ELSE 0 END) AS returned_orders FROM orders GROUP BY 1 ORDER BY 1
```
