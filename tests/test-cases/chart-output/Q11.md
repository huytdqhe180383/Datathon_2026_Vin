# chart-Q11: Plot the average number of days between the first and second order by first_order_month. Return a line chart with first_order_month on the x-axis.

## Question

Plot the average number of days between the first and second order by first_order_month. Return a line chart with first_order_month on the x-axis.

## Reference SQL

```sql
WITH sequenced_orders AS (SELECT customer_id, CAST(order_date AS DATE) AS order_date, ROW_NUMBER() OVER (PARTITION BY customer_id ORDER BY CAST(order_date AS DATE), order_id) AS rn FROM orders), first_two AS (SELECT customer_id, MAX(CASE WHEN rn = 1 THEN order_date END) AS first_order_date, MAX(CASE WHEN rn = 2 THEN order_date END) AS second_order_date FROM sequenced_orders WHERE rn <= 2 GROUP BY 1) SELECT strftime(first_order_date, '%Y-%m') AS first_order_month, ROUND(AVG(date_diff('day', first_order_date, second_order_date)), 4) AS avg_days_first_to_second_order FROM first_two WHERE second_order_date IS NOT NULL GROUP BY 1 ORDER BY 1
```
