## Question

Plot the distribution of customer lifetime net revenue. Return a histogram based on lifetime_net_revenue.

## Reference SQL

```sql
WITH customer_revenue AS (SELECT o.customer_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS lifetime_net_revenue FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1) SELECT customer_id, lifetime_net_revenue FROM customer_revenue ORDER BY lifetime_net_revenue
```
