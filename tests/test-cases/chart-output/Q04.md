## Question

Plot delivered net revenue by age_group and gender. Return a grouped bar chart with age_group on the x-axis.

## Reference SQL

```sql
SELECT c.age_group, c.gender, ROUND(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END), 2) AS delivered_net_revenue FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) GROUP BY 1, 2 ORDER BY age_group, gender
```
