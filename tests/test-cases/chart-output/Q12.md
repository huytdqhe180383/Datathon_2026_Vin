## Question

Plot the top 20 product_id values by delivered units. Return a bar chart using product_id and delivered_units.

## Reference SQL

```sql
SELECT oi.product_id, SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity ELSE 0 END) AS delivered_units FROM order_items oi JOIN orders o USING (order_id) GROUP BY 1 HAVING SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity ELSE 0 END) > 0 ORDER BY delivered_units DESC, product_id LIMIT 20
```
