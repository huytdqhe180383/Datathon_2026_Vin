## Question

Plot discount_rate_pct by device_type. Return a bar chart with device_type on the x-axis.

## Reference SQL

```sql
SELECT o.device_type, ROUND(100.0 * SUM(oi.discount_amount) / NULLIF(SUM(oi.quantity * oi.unit_price), 0), 4) AS discount_rate_pct FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1 ORDER BY discount_rate_pct DESC, device_type
```
