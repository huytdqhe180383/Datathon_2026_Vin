## Question

Plot the delivered weighted average selling price per unit after discounts by month. Return a line chart using order_month and weighted_avg_selling_price_per_unit.

## Reference SQL

```sql
SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, ROUND(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END) / NULLIF(SUM(CASE WHEN o.order_status = 'delivered' THEN oi.quantity ELSE 0 END), 0), 4) AS weighted_avg_selling_price_per_unit FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1 ORDER BY 1
```
