## Question

Plot a cumulative customer revenue curve. Return a line chart using percentile_bucket and cumulative_revenue_share_pct.

## Reference SQL

```sql
WITH customer_revenue AS (SELECT o.customer_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS lifetime_net_revenue FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1), bucketed AS (SELECT NTILE(100) OVER (ORDER BY lifetime_net_revenue DESC, customer_id) AS percentile_bucket, lifetime_net_revenue FROM customer_revenue), bucket_totals AS (SELECT percentile_bucket, SUM(lifetime_net_revenue) AS revenue_in_bucket FROM bucketed GROUP BY 1), cumulative AS (SELECT percentile_bucket, SUM(revenue_in_bucket) OVER (ORDER BY percentile_bucket) AS cumulative_revenue, SUM(revenue_in_bucket) OVER () AS total_revenue FROM bucket_totals) SELECT percentile_bucket, ROUND(100.0 * cumulative_revenue / total_revenue, 4) AS cumulative_revenue_share_pct FROM cumulative ORDER BY percentile_bucket
```
