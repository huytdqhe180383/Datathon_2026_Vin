# value-Q14: Which device type has the highest discount rate, and what is that rate?

## Question

Which device type has the highest discount rate, and what is that rate?

## Reference SQL

```sql
SELECT o.device_type, ROUND(100.0 * SUM(oi.discount_amount) / SUM(oi.quantity * oi.unit_price), 4) AS discount_rate_pct FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Group by device_type. The discount rate is the sum of discount_amount divided by the gross revenue (quantity * unit_price). Sort descending and return the top device type.
