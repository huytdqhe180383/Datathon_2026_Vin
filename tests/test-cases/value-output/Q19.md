# value-Q19: Which order source has the lowest delivered rate, and what is that rate?

## Question

Which order source has the lowest delivered rate, and what is that rate?

## Reference SQL

```sql
SELECT order_source, ROUND(100.0 * AVG(CASE WHEN order_status = 'delivered' THEN 1.0 ELSE 0.0 END), 4) AS delivered_rate_pct FROM orders GROUP BY 1 ORDER BY 2 ASC, 1 LIMIT 1
```

## Explanation

Group by order_source. Calculate the delivered rate as the average of a 1/0 flag (1 if 'delivered', else 0). Sort ascending (ASC) and take the first row to find the lowest rate.
