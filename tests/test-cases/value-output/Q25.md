# value-Q25: What percentage of delivered net revenue came from female customers?

## Question

What percentage of delivered net revenue came from female customers?

## Reference SQL

```sql
SELECT ROUND(100.0 * SUM(CASE WHEN c.gender = 'female' THEN oi.quantity * oi.unit_price - oi.discount_amount ELSE 0 END) / SUM(oi.quantity * oi.unit_price - oi.discount_amount), 4) AS female_delivered_revenue_share_pct FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered'
```

## Explanation

Filter for 'delivered' orders. Sum the net revenue strictly for female customers, divide it by the total net revenue of all customers, and multiply by 100 to get the percentage.
