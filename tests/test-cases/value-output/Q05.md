# value-Q05: Which age group has the highest delivered average order value, and what is that value?

## Metadata

| --- | --- |

## Question

Which age group has the highest delivered average order value, and what is that value?

## Reference SQL

```sql
WITH order_net_values AS (SELECT o.order_id, c.age_group, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS order_net_value FROM orders o JOIN customers c USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1,2) SELECT age_group, ROUND(AVG(order_net_value), 2) AS avg_order_value FROM order_net_values GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

The CTE 'order_net_values' calculates the net value of each order alongside the customer's age group. The main query groups by age group, averages the order values, sorts descending, and retrieves the highest one.
