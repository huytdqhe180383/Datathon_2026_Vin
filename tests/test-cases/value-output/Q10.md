# value-Q10: Which calendar month had the highest delivered net revenue, and what was the amount?

## Metadata

| Field | Value |
| --- | --- |

| Expected output type | `value` |

## Question

Which calendar month had the highest delivered net revenue, and what was the amount?

## Reference SQL

```sql
SELECT strftime(CAST(o.order_date AS DATE), '%Y-%m') AS order_month, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Use strftime('%Y-%m') to extract the year-month from order_date. Group by this month, sum the net revenue for 'delivered' orders, sort descending, and return the top month.
