# value-Q08: What share of total net revenue comes from orders that used at least one promo code?

## Question

What share of total net revenue comes from orders that used at least one promo code?

## Reference SQL

```sql
WITH order_promo_flags AS (SELECT order_id, MAX(CASE WHEN promo_id <> 'NO_PROMO' OR COALESCE(NULLIF(TRIM(CAST(promo_id_2 AS VARCHAR)), ''), 'NO_PROMO') <> 'NO_PROMO' THEN 1 ELSE 0 END) AS has_promo FROM order_items GROUP BY 1), revenue_by_order AS (SELECT order_id, SUM(quantity * unit_price - discount_amount) AS order_net_revenue FROM order_items GROUP BY 1) SELECT ROUND(100.0 * SUM(CASE WHEN p.has_promo = 1 THEN r.order_net_revenue ELSE 0 END) / SUM(r.order_net_revenue), 4) AS promo_revenue_share_pct FROM revenue_by_order r JOIN order_promo_flags p USING (order_id)
```

## Explanation

The 'order_promo_flags' CTE checks if any item in an order used a promo code (promo_id or promo_id_2 is not 'NO_PROMO'). The main query then divides the total revenue of promo-applied orders by the overall total revenue.
