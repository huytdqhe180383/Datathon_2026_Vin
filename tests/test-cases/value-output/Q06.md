# value-Q06: What is the 90th percentile of delivered order net value?

## Question

What is the 90th percentile of delivered order net value?

## Reference SQL

```sql
WITH order_net_values AS (SELECT o.order_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS order_net_value FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1) SELECT ROUND(quantile_cont(order_net_value, 0.9), 2) AS p90_delivered_order_value FROM order_net_values
```

## Explanation

Calculate the net value of each delivered order in the CTE. Then use the quantile_cont(..., 0.9) function to find the 90th percentile of these order values.
