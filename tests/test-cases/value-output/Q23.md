## Question

What is the average number of units per delivered order?

## Reference SQL

```sql
WITH delivered_order_units AS (SELECT o.order_id, SUM(oi.quantity) AS units_per_order FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1) SELECT ROUND(AVG(units_per_order), 4) AS avg_units_per_delivered_order FROM delivered_order_units
```

## Explanation

The CTE sums the quantity of items for each 'delivered' order. The main query then calculates the average number of units per order from this CTE.
