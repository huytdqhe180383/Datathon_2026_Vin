# value-Q17: Among customers whose first order was in their signup month, what is the average first-order net value?

## Question

Among customers whose first order was in their signup month, what is the average first-order net value?

## Reference SQL

```sql
WITH first_orders AS (SELECT o.customer_id, o.order_id, CAST(o.order_date AS DATE) AS first_order_date, ROW_NUMBER() OVER (PARTITION BY o.customer_id ORDER BY CAST(o.order_date AS DATE), o.order_id) AS rn FROM orders o), first_order_values AS (SELECT f.customer_id, f.first_order_date, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS first_order_net_value FROM first_orders f JOIN order_items oi USING (order_id) WHERE f.rn = 1 GROUP BY 1,2) SELECT ROUND(AVG(first_order_net_value), 2) AS avg_first_order_value_same_signup_month FROM first_order_values f JOIN customers c USING (customer_id) WHERE strftime(CAST(c.signup_date AS DATE), '%Y-%m') = strftime(f.first_order_date, '%Y-%m')
```

## Explanation

Use ROW_NUMBER() to identify each customer's first order date. Filter for customers where the month of the first order matches their signup month (using strftime). Calculate the average value of these first orders.
