# value-Q26: Among zip codes with at least 100 delivered orders, which zip code has the highest average delivered order value, and what is that value?

## Question

Among zip codes with at least 100 delivered orders, which zip code has the highest average delivered order value, and what is that value?

## Reference SQL

```sql
WITH delivered_order_values AS (SELECT o.zip, o.order_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS order_net_value FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1,2), zip_stats AS (SELECT zip, COUNT(*) AS delivered_order_count, AVG(order_net_value) AS avg_order_value FROM delivered_order_values GROUP BY 1) SELECT zip, delivered_order_count, ROUND(avg_order_value, 2) AS avg_order_value FROM zip_stats WHERE delivered_order_count >= 100 ORDER BY avg_order_value DESC, zip LIMIT 1
```

## Explanation

Calculate the value of each order by zip code. Group by zip to count the number of delivered orders and average order value. Filter for zip codes with >= 100 orders, sort descending by average value.
