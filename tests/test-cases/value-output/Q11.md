# value-Q11: What is the median number of days between consecutive orders for repeat customers?

## Question

What is the median number of days between consecutive orders for repeat customers?

## Reference SQL

```sql
WITH ordered_dates AS (SELECT customer_id, CAST(order_date AS DATE) AS order_date, LAG(CAST(order_date AS DATE)) OVER (PARTITION BY customer_id ORDER BY CAST(order_date AS DATE)) AS prev_order_date FROM orders), order_gaps AS (SELECT date_diff('day', prev_order_date, order_date) AS gap_days FROM ordered_dates WHERE prev_order_date IS NOT NULL) SELECT ROUND(median(gap_days), 4) AS median_days_between_orders FROM order_gaps
```

## Explanation

Use LAG() to get the date of the previous order for the same customer. Calculate the date_diff in days between the current and previous order. Use the median() function to find the median of these gaps.
