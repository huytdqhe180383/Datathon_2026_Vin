# value-Q07: Which payment method has the highest return rate, and what is that rate?

## Question

Which payment method has the highest return rate, and what is that rate?

## Reference SQL

```sql
SELECT payment_method, ROUND(100.0 * AVG(CASE WHEN order_status = 'returned' THEN 1.0 ELSE 0.0 END), 4) AS return_rate_pct FROM orders GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Group by payment_method. The return rate is calculated by taking the average of a 1/0 flag: 1 if status is 'returned', 0 otherwise. Multiply by 100 for percentage, order descending, and get the top row.
