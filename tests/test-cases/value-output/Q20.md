# value-Q20: What is the average number of distinct purchase months for repeat customers?

## Question

What is the average number of distinct purchase months for repeat customers?

## Reference SQL

```sql
WITH customer_months AS (SELECT customer_id, COUNT(DISTINCT strftime(CAST(order_date AS DATE), '%Y-%m')) AS distinct_purchase_months, COUNT(*) AS order_count FROM orders GROUP BY 1) SELECT ROUND(AVG(distinct_purchase_months), 4) AS avg_distinct_purchase_months_repeat_customers FROM customer_months WHERE order_count >= 2
```

## Explanation

Count the distinct purchase months (using strftime '%Y-%m') for each customer. Filter for customers with at least 2 orders (repeat customers). Calculate the average number of distinct months for this group.
