# value-Q03: What is the average number of days between a customer's first and second order?

## Question

What is the average number of days between a customer's first and second order?

## Reference SQL

```sql
WITH sequenced_orders AS (SELECT customer_id, CAST(order_date AS DATE) AS order_date, ROW_NUMBER() OVER (PARTITION BY customer_id ORDER BY CAST(order_date AS DATE), order_id) AS rn FROM orders), first_two AS (SELECT customer_id, MAX(CASE WHEN rn = 1 THEN order_date END) AS first_order_date, MAX(CASE WHEN rn = 2 THEN order_date END) AS second_order_date FROM sequenced_orders WHERE rn <= 2 GROUP BY 1) SELECT ROUND(AVG(date_diff('day', first_order_date, second_order_date)), 4) AS avg_days_first_to_second_order FROM first_two WHERE second_order_date IS NOT NULL
```

## Explanation

Use the ROW_NUMBER() window function to chronologically rank orders for each customer. Extract the first order date (rn=1) and second order date (rn=2). Finally, calculate the average difference in days (date_diff) between these two dates.
