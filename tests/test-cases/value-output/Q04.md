# value-Q04: Which acquisition channel generated the highest delivered net revenue, and what was the amount?

## Question

Which acquisition channel generated the highest delivered net revenue, and what was the amount?

## Reference SQL

```sql
SELECT c.acquisition_channel, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Join customers, orders, and order_items. Filter for 'delivered' orders. Group by acquisition_channel and sum the net revenue. Order descending by revenue and use LIMIT 1 to find the top channel.
