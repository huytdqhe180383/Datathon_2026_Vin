# value-Q24: Which acquisition channel has the highest delivered weighted average selling price per unit, and what is that price?

## Question

Which acquisition channel has the highest delivered weighted average selling price per unit, and what is that price?

## Reference SQL

```sql
SELECT c.acquisition_channel, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount) / SUM(oi.quantity), 4) AS weighted_avg_selling_price_per_unit FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Group by acquisition_channel. Calculate the weighted average selling price (total net revenue divided by total quantity) for 'delivered' orders. Sort descending and return the highest channel.
