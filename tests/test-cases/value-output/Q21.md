## Question

What percentage of customers never placed an order?

## Reference SQL

```sql
SELECT ROUND(100.0 * SUM(CASE WHEN o.customer_id IS NULL THEN 1 ELSE 0 END) / COUNT(*), 4) AS customers_without_orders_pct FROM customers c LEFT JOIN (SELECT DISTINCT customer_id FROM orders) o USING (customer_id)
```

## Explanation

Perform a LEFT JOIN from customers to orders. Customers who never ordered will have a NULL customer_id in the orders table. Divide the count of these NULLs by the total number of customers.
