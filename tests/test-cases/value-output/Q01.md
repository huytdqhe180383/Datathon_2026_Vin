# value-Q01: What is the delivered net revenue after discounts?

| --- | --- |

## Question

What is the delivered net revenue after discounts?

## Reference SQL

```sql
SELECT ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered'
```

## Explanation

Join orders and order_items tables. Filter for 'delivered' status. Net revenue is calculated as sum of (quantity * unit_price - discount_amount). ROUND(..., 2) formats the result to 2 decimal places.
