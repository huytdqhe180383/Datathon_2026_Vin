# value-Q28: Which payment method has the highest delivered average order value, and what is that value?

## Metadata

| Field | Value |
| --- | --- |
| Case ID | `value-Q28` |
| Expected output type | `value` |

## Question

Which payment method has the highest delivered average order value, and what is that value?

## Reference SQL

```sql
WITH delivered_order_values AS (SELECT o.payment_method, o.order_id, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS order_net_value FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1,2) SELECT payment_method, ROUND(AVG(order_net_value), 2) AS avg_order_value FROM delivered_order_values GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Group by payment_method and order_id to calculate the value of each order. Then, average the order values per payment method. Sort descending and return the highest result.
