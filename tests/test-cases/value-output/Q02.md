## Question

What percentage of ordering customers placed more than one order?

## Reference SQL

```sql
WITH customer_order_counts AS (SELECT customer_id, COUNT(*) AS order_count FROM orders GROUP BY 1) SELECT ROUND(100.0 * SUM(CASE WHEN order_count > 1 THEN 1 ELSE 0 END) / COUNT(*), 4) AS repeat_customer_rate_pct FROM customer_order_counts
```

## Explanation

Use a Common Table Expression (CTE) 'customer_order_counts' to count orders per customer. Then calculate the percentage: sum of customers with order_count > 1 divided by the total number of customers, multiplied by 100.
