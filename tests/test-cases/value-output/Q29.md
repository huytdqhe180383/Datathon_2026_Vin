## Question

What percentage of total quantity sold came from returned orders?

## Reference SQL

```sql
SELECT ROUND(100.0 * SUM(CASE WHEN o.order_status = 'returned' THEN oi.quantity ELSE 0 END) / SUM(oi.quantity), 4) AS returned_quantity_share_pct FROM orders o JOIN order_items oi USING (order_id)
```

## Explanation

Sum the quantity of items specifically from orders with a 'returned' status, divide it by the total quantity across all orders, and multiply by 100 to get the percentage.
