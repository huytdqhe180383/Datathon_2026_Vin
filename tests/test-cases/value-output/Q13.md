# value-Q13: What percentage of delivered orders contained more than one distinct order-item line?

## Question

What percentage of delivered orders contained more than one distinct order-item line?

## Reference SQL

```sql
WITH delivered_order_line_counts AS (SELECT o.order_id, COUNT(*) AS line_count FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1) SELECT ROUND(100.0 * AVG(CASE WHEN line_count > 1 THEN 1.0 ELSE 0.0 END), 4) AS multi_line_delivered_order_pct FROM delivered_order_line_counts
```

## Explanation

The CTE counts the number of lines (items) in each delivered order. The main query takes the average of a condition (1 if line_count > 1, else 0) and multiplies by 100 to get the percentage.
