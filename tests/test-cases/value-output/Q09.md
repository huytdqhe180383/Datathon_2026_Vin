# value-Q09: Among customers with at least 3 orders, what is the average lifetime net revenue per customer?

## Question

Among customers with at least 3 orders, what is the average lifetime net revenue per customer?

## Reference SQL

```sql
WITH customer_revenue AS (SELECT o.customer_id, COUNT(DISTINCT o.order_id) AS order_count, SUM(oi.quantity * oi.unit_price - oi.discount_amount) AS lifetime_net_revenue FROM orders o JOIN order_items oi USING (order_id) GROUP BY 1) SELECT ROUND(AVG(lifetime_net_revenue), 2) AS avg_lifetime_net_revenue_3plus_orders FROM customer_revenue WHERE order_count >= 3
```

## Explanation

Group by customer_id to count distinct orders and calculate lifetime net revenue per customer. The main query filters for customers with >= 3 orders and calculates their average lifetime revenue.
