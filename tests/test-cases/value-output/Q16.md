# value-Q16: Which acquisition channel has the highest repeat-customer rate, and what is that rate?

## Question

Which acquisition channel has the highest repeat-customer rate, and what is that rate?

## Reference SQL

```sql
WITH channel_customer_orders AS (SELECT c.acquisition_channel, o.customer_id, COUNT(*) AS order_count FROM customers c JOIN orders o USING (customer_id) GROUP BY 1,2) SELECT acquisition_channel, ROUND(100.0 * AVG(CASE WHEN order_count > 1 THEN 1.0 ELSE 0.0 END), 4) AS repeat_customer_rate_pct FROM channel_customer_orders GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Count orders per customer for each acquisition channel. Calculate the repeat-customer rate (customers with >= 2 orders divided by total customers) for each channel. Sort descending to get the highest.
