# value-Q15: What is the delivered weighted average selling price per unit after discounts?

## Question

What is the delivered weighted average selling price per unit after discounts?

## Reference SQL

```sql
SELECT ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount) / SUM(oi.quantity), 4) AS delivered_weighted_avg_selling_price_per_unit FROM orders o JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered'
```

## Explanation

Filter for 'delivered' orders. Divide the total net revenue (after discounts) by the total quantity sold. This gives the weighted average selling price per unit.
