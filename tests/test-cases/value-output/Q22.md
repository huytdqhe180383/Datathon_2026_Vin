# value-Q22: Which age group generated the highest delivered net revenue, and what was the amount?

## Question

Which age group generated the highest delivered net revenue, and what was the amount?

## Reference SQL

```sql
SELECT c.age_group, ROUND(SUM(oi.quantity * oi.unit_price - oi.discount_amount), 2) AS delivered_net_revenue FROM customers c JOIN orders o USING (customer_id) JOIN order_items oi USING (order_id) WHERE o.order_status = 'delivered' GROUP BY 1 ORDER BY 2 DESC, 1 LIMIT 1
```

## Explanation

Similar to question 4, but groups by age_group instead of acquisition_channel. Calculates total net revenue for 'delivered' orders, sorts descending, and returns the highest age group.
